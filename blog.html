Discuss the scope of var, let, and const

  • var: Basically, we are using var for declare a variable. Its initial value is undefined. It’s hoisted on the top and it’s a global scop.
  • let: Let use to declare variable, and if we want, we can reassign its variable. let are block-scoped.
  • const: Const is similar with let but it is immutable, means their values cannot be reassigned.

Tell us the use cases of null and undefined

  • null: We use null when we intentionally set a variable empty, we don’t know the value.
  • undefined: Variables that are declared but not assigned a value are automatically initialized with undefined. This is the default value for variables.
